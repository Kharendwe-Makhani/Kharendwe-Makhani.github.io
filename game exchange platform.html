Game Exchange Platform

During one of my vacation work periods at AppChemy, I was tasked with creating a platform or website where people can exchange video games. That is, if Gamer A has Game X, they can exchange Game X for Game Y with Gamer B.

The design of such a platform would require a software developer, I am only a mechatronic engineer so there was only much that I could accomplish. So instead of designing a fully functional platform, I only created the skeleton of what more or less the platform will look like.

I ended up only being able to create the sign-up screen and the layout of the user navigation screen. But it was still mind-opening to try something outside of the mechatronic engineering field.
